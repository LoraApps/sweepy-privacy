Sweepy — Kebijakan Privasi

Perilaku aplikasi saat ini dan detail pemrosesan data
Diperbarui: 12 Januari 2026

Ringkasan

Sweepy ("Aplikasi") adalah alat untuk membersihkan galeri dan mengelola foto/video yang berjalan di perangkat Anda. Kebijakan ini menjelaskan izin yang digunakan, data apa saja yang dapat diproses, layanan pihak ketiga yang terlibat, serta hak Anda.

Data yang Diproses dan Alasannya

Secara umum, Sweepy tidak mengunggah konten media pribadi Anda ke server. Namun, beberapa jenis data berikut dapat diproses oleh aplikasi dan/atau layanan pihak ketiga:

  • Akses media: Untuk menampilkan dan mengelola foto/video Anda (melihat, kompresi, penghapusan, Vault pribadi). File media tetap di perangkat; kami tidak mengunggah isi file ke server.
  • Pengaturan & penghitung (lokal): Data kecil seperti preferensi, penghitung harian, dan cache disimpan secara lokal di Hive (database lokal).
  • Analitik & laporan crash: Aplikasi dapat mengumpulkan metrik penggunaan dan laporan crash melalui Firebase Analytics atau alat serupa. Ini untuk stabilitas dan performa, dan tidak mencakup isi media pribadi Anda.
  • Pengidentifikasi iklan: Agar iklan berfungsi, Advertising ID perangkat dan sinyal kontekstual dapat digunakan; pemrosesan dilakukan oleh Google Mobile Ads.
  • Pembayaran / langganan: Pembelian diproses melalui RevenueCat (purchases_flutter). Informasi seperti status langganan diproses oleh RevenueCat; tidak dikirim ke server kami sendiri.

Izin dan Penggunaannya

Aplikasi dapat meminta izin Android berikut. Setiap izin hanya digunakan saat fitur terkait diaktifkan:

  • READ_MEDIA_IMAGES / READ_MEDIA_VIDEO: Diperlukan untuk menampilkan dan memproses media di galeri.
  • WRITE_EXTERNAL_STORAGE (API lama): Digunakan untuk fitur yang membutuhkan menulis/menghapus file (Android 9 ke bawah).
  • INTERNET: Diperlukan untuk penayangan iklan, pemeriksaan konfigurasi/paket, dan layanan pembelian.

Layanan Pihak Ketiga

Beberapa fitur menggunakan SDK dan layanan pihak ketiga yang diatur oleh kebijakan privasi masing-masing:

Google Mobile Ads (AdMob)
RevenueCat (purchases_flutter)
Firebase (Analytics / Crashlytics)
Hive (data lokal)
photo_manager (akses galeri)

Layanan ini dapat mengumpulkan data seperti informasi pribadi atau pengidentifikasi iklan. Misalnya, penyedia iklan dapat menggunakan Advertising ID untuk penargetan. Silakan merujuk ke kebijakan privasi penyedia untuk detail.

Iklan dan Personalisasi

Aplikasi menampilkan iklan. Penyedia iklan (mis. Google) dapat mempersonalisasi iklan berdasarkan pengidentifikasi perangkat, lokasi (jika diizinkan), dan penggunaan aplikasi. Anda dapat mengatur preferensi iklan melalui pengaturan iklan Android atau pengaturan perangkat.

Pembelian dan Langganan

Pembelian dalam aplikasi diproses melalui RevenueCat. Saat pembelian, informasi pembayaran ditangani langsung oleh store (Google Play / App Store); Aplikasi dan RevenueCat tidak menerima detail pembayaran lengkap. Status pembelian/langganan Anda disimpan dan diberlakukan oleh RevenueCat.

Penyimpanan Lokal dan Cache

Aplikasi menyimpan sejumlah kecil data di perangkat Anda (Hive): preferensi, penghitung harian, cache kecil, dan status aplikasi. Data ini biasanya terhapus saat Anda menghapus aplikasi, namun perilaku dapat berbeda tergantung perangkat/OS.

Retensi dan Penghapusan Data

  • Aplikasi tidak menyimpan salinan media Anda di server.
  • File yang Anda hapus di aplikasi akan dihapus dari perangkat; perilaku "trash"/"recycle" Android dapat berbeda.
  • Untuk menghapus data lokal (pengaturan, penghitung), hapus aplikasi; dalam beberapa kasus Anda perlu menghapus data aplikasi secara manual (Pengaturan perangkat → Aplikasi).

Hak Anda

Karena sebagian besar data disimpan secara lokal, umumnya tidak ada permintaan ke server pusat yang diperlukan. Untuk data yang dikumpulkan oleh penyedia iklan/analitik, Anda dapat mengelola preferensi dan permintaan penghapusan melalui kontrol pihak ketiga (mis. pengaturan Google).

Privasi Anak

Aplikasi tidak dengan sengaja mengumpulkan data pribadi dari anak di bawah 13 tahun. Jika Anda orang tua/wali dan yakin data telah dikumpulkan, silakan hubungi kami.

Keamanan

Aplikasi menerapkan langkah keamanan yang wajar untuk data lokal (mis. enkripsi untuk Vault pribadi di aplikasi). Saat ada transmisi jaringan (verifikasi pembelian, iklan), praktik keamanan penyedia pihak ketiga berlaku.
